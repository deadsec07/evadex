**Security Policy**

Scope and intent:

- EvadeX is an educational simulation. It is not intended for operational use in any physical, safety-critical, or weapons-related system.
- The maintainers do not accept feature requests or contributions that enable real-world targeting, evasion, or deployment.

Reporting a vulnerability:

- If you discover a security issue in the repository (e.g., a supply-chain or code execution concern), please open a private issue or contact the maintainers.
- Do not disclose publicly until a reasonable mitigation plan is in place.

Out-of-scope:

- Guidance or support for integrating this code with hardware, communications equipment, or military systems.
